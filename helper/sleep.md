`sleep`

#### Mammal sleep data

#### Description

Sleep data with missing values.

#### **Format**

A data frame with 62 observations on the following 10 variables.

**BodyWgt**: 
a numeric vector

**BrainWgt**:
a numeric vector

**NonD**: 
a numeric vector

**Dream:**
a numeric vector

**Sleep:**
a numeric vector

**Span:**
a numeric vector

**Gest:**
a numeric vector

**Pred:**
a numeric vector

**Exp:**
a numeric vector

**Danger:**
a numeric vector
